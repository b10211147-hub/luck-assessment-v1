# 運勢改善健檢｜完全免費第一版

這是一個純前端、零依賴、零資料庫、零主機費的運勢改善健檢 Web App，提供四種 17 題主題短版與一個 65 題完整版。

## 立即使用

1. 解壓縮專案。
2. 雙擊 `index.html`。
3. 使用 Chrome、Edge、Safari 或 Firefox 開啟。

不需要安裝 Node.js，也不需要任何帳號。

## 已完成

- 首頁可選感情、事業、財運、家運或完整版
- 四種主題短版各 17 題，完整版 65 題
- 手機與電腦版面
- 各版本分開自動儲存瀏覽器進度
- 四軸 A/R、I/E、W/L、B/S 計分
- 四碼類型與比例
- 八大面向分數與排名
- 一致性檢核
- 服務適用性提醒
- 客人版初步結果說明
- 內部檢核資料保留於程式結果資料中，預留未來收件串接

## 完全免費部署方式

### GitHub Pages

1. 在 GitHub 建立公開 repository。
2. 上傳本專案全部檔案。
3. Repository Settings → Pages。
4. Source 選 `Deploy from a branch`。
5. Branch 選 `main` 與 `/root`。

公開網址會是：

`https://你的帳號.github.io/專案名稱/`

GitHub Pages 適合純前端靜態網站。此專案不傳送或集中保存客戶資料，答案只存在受測者自己的瀏覽器。

## 隱私限制

完全免費版本沒有中央資料庫，因此你無法自動在後台看到客戶答案。目前客人只會看到主要類型、目前困境與初步改善建議。

若未來要集中收件，可加入 Google Apps Script + Google Sheet。前端已預留送出結果功能，設定完成後會送出客人看到的類型描述、目前困境、大概可以怎麼改善，以及內部用的推薦科儀項目、八大面向、檢核與完整答案。

## Google Sheet 收件設定

1. 建立一個新的 Google Sheet。
2. 在 Google Sheet 選「擴充功能」→「Apps Script」。
3. 將本專案的 `google-apps-script.gs` 內容貼到 Apps Script。
4. 按「部署」→「新增部署作業」。
5. 類型選「網頁應用程式」。
6. 執行身分選「我」。
7. 存取權選「任何人」。
8. 部署並授權後，複製 `/exec` 結尾的 Web App 網址。
9. 將該網址填入 `app.js` 的 `GOOGLE_SCRIPT_URL`。

設定完成後，客人完成測驗時會看到送出結果按鈕；送出後資料會寫入 Google Sheet 的「測驗結果」工作表，並記錄測驗版本與題數。`推薦科儀項目` 欄位只作為內部初步參考，仍需人工判斷與後續評估。

## 修改題目

所有題目與計分設定都在 `app.js`：

- `pairQuestions`：第 1～32 題
- `singleQuestions`：第 33～56 題
- `checkQuestions`：第 57～60 題
- `safetyQuestions`：第 61～65 題
- `quizModes`：五種測驗版本的題目組合與相關面向

短版使用第 1～8 題作為四軸核心題，再搭配三個相關面向各 3 題，共 17 題。短版四軸以平均分數換算比例，因此結果會標示為初步類型；完整版仍沿用全部題目與完整檢核。

此工具只做近期狀態與運勢傾向盤點，不應用來直接判定祖先、靈性、因果或醫療問題，也不應保證科儀效果。
